# The AI Signal

A scalable Next.js 14 starter built with TypeScript, Tailwind CSS, and the App Router.

## Structure

- app/ for routes and layout
- components/ for shared UI
- lib/ for utilities
- services/ for API logic
- types/ for shared TypeScript types
- hooks/ for reusable hooks

## Scripts

- npm run dev
- npm run build
- npm run start
- npm run lint
